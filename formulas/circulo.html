Fórmulas del Circulo

Área del Círculo

A = πr²

Donde:

  • A: Área
  • π: Constante Pi (~3.1416)
  • r: Radio

¿Cómo se obtiene?

El área de un círculo es proporcional al cuadrado de su radio. La constante π (pi) es la relación entre la circunferencia de un círculo y su diámetro.

Ejemplo práctico

Para un círculo con radio 4:

A = π × 4² ≈ 50.27 unidades²

Circunferencia (Perímetro)

C = 2πr

Donde:

  • C: Circunferencia
  • π: Constante Pi (~3.1416)
  • r: Radio

¿Cómo se obtiene?

La circunferencia es directamente proporcional al radio. Como π es la relación entre circunferencia y diámetro, multiplicamos por 2 para obtener la relación con el radio.

Ejemplo práctico

Para el mismo círculo de radio 4:

C = 2 × π × 4 ≈ 25.13 unidades
