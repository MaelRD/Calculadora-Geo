Fórmulas del Triangulo

Área del Triángulo

A = (b × h) / 2

Donde:

  • A: Área
  • b: Base
  • h: Altura

¿Cómo se obtiene?

El área de un triángulo es igual a la mitad del producto de su base por su altura. Esto se debe a que un triángulo es la mitad de un paralelogramo con la misma base y altura.

Ejemplo práctico

Para un triángulo con base 6 y altura 4:

A = (6 × 4) / 2 = 12 unidades²

Perímetro del Triángulo

P = L1 + L2 + L3

Donde:

  • P: Perímetro
  • L1, L2, L3: Lados del triángulo

¿Cómo se obtiene?

El perímetro es la suma de las longitudes de sus tres lados. Esta fórmula aplica para cualquier tipo de triángulo (equilátero, isósceles o escaleno).

Ejemplo práctico

Para un triángulo con lados 3, 4 y 5:

P = 3 + 4 + 5 = 12 unidades
